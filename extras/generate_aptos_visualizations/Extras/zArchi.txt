                  ┌──────────────────────────────┐
                  │  Ground Truth Targets        │
                  ├──────────────────────────────┤
                  │ {                            │
                  │   'ordinal': (B,) ∈ {0..4},  │
                  │   'binary': (B,) ∈ {0, 1},   │

                  ┌──────────────┼──────────────┐
                  │              │              │
                  ▼              ▼              ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                 MULTI-TASK LOSS (losses.py - MultiTaskLoss)                 │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  L_total = L_ord + 0.3·L_bin + 0.2·L_reg + λ_qwk·L_qwk + λ_cons·L_cons     │
│                                                                              │
│  ┌────────────────────────────────────────────────────────────────┐        │
│  │ 1. ORDINAL LOSS (corn_loss)                     Weight: 1.0    │        │
│  │ ────────────────────────────────────────────────────────────   │        │
│  │ CORN (Conditional Ordinal Regression)                          │        │
│  │                                                                 │        │
│  │ For 5 classes, create 4 binary thresholds:                     │        │
│  │ • Threshold 1: P(grade > 0)                                    │        │
│  │ • Threshold 2: P(grade > 1)                                    │        │
│  │ • Threshold 3: P(grade > 2)                                    │        │
│  │ • Threshold 4: P(grade > 3)                                    │        │
│  │                                                                 │        │
│  │ Loss = BCE_with_logits(logits, binary_targets,                │        │
│  │                        pos_weight=1.5) + label_smoothing       │        │
│  │                                                                 │        │
│  │ pos_weight=1.5: Emphasize positive cases (minority classes)   │        │
│  │ label_smoothing=0.05: Prevent overconfidence                  │        │
│  └─────────────────────────────────────────────────────────────────┘        │
│                                                                              │
│  ┌────────────────────────────────────────────────────────────────┐        │
│  │ 2. BINARY LOSS (focal_loss_binary)              Weight: 0.3    │        │
│  │ ────────────────────────────────────────────────────────────   │        │
│  │ Focal Loss for Referable DR (Grade ≥ 2)                       │        │
│  │                                                                 │        │
│  │ FL(p) = -α·(1-p_t)^γ·BCE(p, y)                                │        │
│  │                                                                 │        │
│  │ Parameters:                                                     │        │
│  │ • α (alpha) = 0.40 (minority class weight)                    │        │
│  │ • γ (gamma) = 2.0 (focus on hard examples)                    │        │
│  │                                                                 │        │
│  │ Purpose: Handle severe class imbalance in referable DR         │        │
│  └─────────────────────────────────────────────────────────────────┘        │
│                                                                              │
│  ┌────────────────────────────────────────────────────────────────┐        │
│  │ 3. REGRESSION LOSS (Huber Loss)                  Weight: 0.2   │        │
│  │ ────────────────────────────────────────────────────────────   │        │
│  │ Smooth L1 Loss (robust to outliers)                            │        │
│  │                                                                 │        │
│  │         { 0.5·(pred - true)²       if |pred-true| ≤ δ          │        │
│  │ L(p,t)= {                                                      │        │
│  │         { δ·|pred-true| - 0.5·δ²  otherwise                   │        │
│  │                                                                 │        │
│  │ δ (delta) = 1.0                                                │        │
│  │                                                                 │        │
│  │ Purpose: Smooth regression guidance for severity score         │        │
│  └─────────────────────────────────────────────────────────────────┘        │
│                                                                              │
│  ┌────────────────────────────────────────────────────────────────┐        │
│  │ 4. QWK LOSS (QWKLoss)                          Weight: λ_qwk   │        │
│  │ ────────────────────────────────────────────────────────────   │        │
│  │ Differentiable Quadratic Weighted Kappa                        │        │
│  │                                                                 │        │
│  │ 1. Convert CORN logits → class probabilities:                  │        │
│  │    p[0] = 1 - σ(logit_0)                                       │        │
│  │    p[1] = σ(logit_0) × (1 - σ(logit_1))                        │        │
│  │    p[2] = σ(logit_0) × σ(logit_1) × (1 - σ(logit_2))          │        │
│  │    p[3] = σ(logit_0) × σ(logit_1) × σ(logit_2) × (1-σ(logit_3))│        │
│  │    p[4] = σ(logit_0) × σ(logit_1) × σ(logit_2) × σ(logit_3)   │        │
│  │                                                                 │        │
│  │ 2. Build soft confusion matrix O                               │        │
│  │ 3. Build expected matrix E                                     │        │
│  │ 4. QWK = 1 - Σ(W·O) / Σ(W·E)                                  │        │
│  │                                                                 │        │
│  │ Weight matrix W[i,j] = (i-j)² / (N-1)²                         │        │
│  │                                                                 │        │
│  │ λ_qwk curriculum:                                              │        │
│  │ • Epochs 1-10:  λ = 0.0   (disabled)                          │        │
│  │ • Epochs 11-30: λ = 0.05  (gentle)                            │        │
│  │ • Epochs 31-50: λ = 0.1   (full weight)                       │        │
│  └─────────────────────────────────────────────────────────────────┘        │
│                                                                              │
│  ┌────────────────────────────────────────────────────────────────┐        │
│  │ 5. CONSISTENCY LOSS (Huber)                   Weight: λ_cons   │        │
│  │ ────────────────────────────────────────────────────────────   │        │
│  │ Enforce agreement between ordinal and regression heads         │        │
│  │                                                                 │        │
│  │ expected_grade = Σ P(grade > i) for i in [0,1,2,3]            │        │
│  │                = σ(logit_0) + σ(logit_1) +                     │        │
│  │                  σ(logit_2) + σ(logit_3)                       │        │
│  │                                                                 │        │
│  │ L_cons = Huber(expected_grade, regression_score, δ=1.0)        │        │
│  │                                                                 │        │
│  │ λ_cons curriculum:                                             │        │
│  │ • Epochs 1-20:  λ = 0.02                                      │        │
│  │ • Epochs 21-50: λ = 0.05                                      │        │
│  └─────────────────────────────────────────────────────────────────┘        │
│                                                                              │

                           🎯 PERFORMANCE METRICS
═══════════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────────┐
│                        PRIMARY METRIC: QWK                                   │
├─────────────────────────────────────────────────────────────────────────────┤
│  Quadratic Weighted Kappa (Cohen's Kappa with quadratic weights)            │
│                                                                              │
│  Measures agreement between predictions and ground truth                    │
│  Accounts for ordinal nature of DR grades                                   │
│  Penalizes larger disagreements more heavily                                │
│                                                                              │
│  Range: [-1, 1]                                                             │
│  • 1.0  = Perfect agreement                                                 │
│  • 0.9+ = Excellent                                                         │
│  • 0.8+ = Good                                                              │
│  • 0.7+ = Moderate                                                          │
│  • <0.7 = Needs improvement                                                 │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│                       SECONDARY METRICS                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│  (metrics.py - calculate_metrics)                                           │
│                                                                              │
│  • Accuracy: Overall classification correctness                             │
│  • Sensitivity (Recall): TP / (TP + FN) - catch positive DR cases          │
│  • Specificity: TN / (TN + FP) - correctly identify healthy                │
│  • Macro F1-Score: Harmonic mean of precision and recall                   │
│  • Per-class Precision/Recall: Performance per severity level               │
│  • Confusion Matrix: Detailed error analysis                                │
└─────────────────────────────────────────────────────────────────────────────┘
